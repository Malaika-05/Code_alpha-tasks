# AI Mini Projects Collection

## 📌 Overview

This repository contains three Artificial Intelligence and Natural Language Processing (NLP) based mini projects developed using Python. The projects demonstrate practical applications of AI including language translation, intelligent question answering, and music generation using deep learning techniques.

---

# 📖 Projects Included

## 1. Language Translation Tool

### Description

The Language Translation Tool allows users to enter text, select the source and target languages, and receive an accurate translation using a translation API.

### Features

* User-friendly interface for text input
* Selection of source and target languages
* Translation using external APIs (Google Translate)
* Displays translated text clearly
* Additional usability features such as copy text and text-to-speech (optional)

### Technologies Used

* Python
* Translation APIs
* GUI Framework (Tkinter/Streamlit/Flask)
* NLP Techniques

---

## 2. FAQ Chatbot

### Description

The FAQ Chatbot provides automated responses to user questions by matching user input with the most relevant frequently asked question using Natural Language Processing techniques.

### Features

* FAQ dataset containing questions and answers
* Text preprocessing (cleaning, tokenization, stopword removal)
* Similarity matching using NLP techniques
* Finds the most relevant answer to user queries
* Interactive chatbot interface

### Technologies Used

* Python
* NLTK / SpaCy
* Scikit-learn
* TF-IDF Vectorization
* Cosine Similarity

---

## 3. AI Music Generation System

### Description

This project generates new musical compositions using Artificial Intelligence by learning patterns from MIDI music datasets.

### Features

* MIDI dataset processing
* Music note extraction and sequence generation
* Deep learning-based music generation
* Creates new melodies and saves them as MIDI/audio files

### Technologies Used

* Python
* Music21
* TensorFlow / Keras
* RNN (LSTM) / GAN Models
* MIDI Processing
